What is DetectSus?

DetectSus is a real-time malpractice detection system designed to identify suspicious activities in classrooms using advanced computer vision and AI. Our solution helps educational institutions strengthen exam integrity by monitoring for unauthorized materials, unnatural gestures, and other cheating behaviors.

  • Object detection for identifying phones, notes, and more
  • Pose estimation to capture suspicious body language
  • Immediate alerts for quick intervention
